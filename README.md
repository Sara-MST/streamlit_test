# Interactive Data Dashboard with Streamlit

## Project overview
This project consists of an interactive web application built with Streamlit
to explore and visualize a dataset of vehicle listings.

## Features
- Interactive histograms
- Scatter plots
- User-controlled visualizations

## Tech stack
- Python
- Pandas
- Plotly Express
- Streamlit

## Deployment
The application is deployed on Render and accessible via a public URL.

## Notes
Developed as part of a Data Analysis bootcamp.
